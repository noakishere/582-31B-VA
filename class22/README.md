## Key Vocabulary

- Model --> A Python class mapped to a database table
- Primary Key --> A value that uniquely identifies a row
- Record --> One row in a database table
- Domain Logic --> Rules and behaviour belonging to domain objects
- Business rule --> A rule controlling what actions are valid

- Foreign Key --> A column that refers to another table's **primary key**
- Relationship --> A Python-level conneciton between model objects

- Parent Model --> The "one" side of a one-to-many relationship
- Child Model --> The "many" side of a one-to-many relationship
